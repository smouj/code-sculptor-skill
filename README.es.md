# Code Sculptor

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Code Sculptor para OpenClaw.

## Qué Soluciona
Reestructura y optimiza automáticamente las bases de código para mejorar su mantenibilidad y rendimiento.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y preparados para producción
- Enfoque prioritario en seguridad
- Soporte para reversión

## Uso
/code-sculptor

## Ejemplo
```
Analyze code for automation issues.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
